10.16 Intro to Data Science¶

Time Series and Simple Linear Regression¶

Time series are sequences of data points recorded over time, often at regular intervals.
We’ll use simple linear regression to explore and predict trends in time series data.

We’ll use time series to:

  1. Analyze patterns in historical data.
  2. Forecast future values using regression.

What Is a Time Series?¶

A time series is a collection of observations recorded at successive points in time.

Examples:

  • Daily closing stock prices
  • Hourly temperature readings
  • Changing positions of a plane in flight
  • Annual crop yields
  • Quarterly company profits

Each observation is associated with a timestamp.

Univariate vs. Multivariate Time Series¶

  • Univariate: One observation per time point
    Example: Average January temperature per year in New York City.
  • Multivariate: Two or more observations per time point
    Example: Temperature, humidity, and barometric pressure recorded together.

Common Time Series Tasks¶

  1. Time Series Analysis

    • Looks for patterns, seasonality, and trends in past data.
    • Example: Monthly temperature variations across seasons.

  2. Time Series Forecasting

    • Uses past data to predict future values.
    • Example: Predicting next year’s average January temperature.

Simple Linear Regression¶

Understanding Linear Relationships with Real Data¶

We’ll use Simple Linear Regression to model and predict the relationship between:

  • Independent variable (x): Year (time)
  • Dependent variable (y): Average January high temperature in NYC

What Is a Linear Relationship?¶

A linear relationship exists when one variable changes proportionally with another.

Example: Fahrenheit and Celsius temperatures are linearly related by

$$ C = \frac{5}{9} (F - 32) $$

Here:

  • F = Fahrenheit temperature (independent variable)
  • C = Celsius temperature (dependent variable)

Components of the Simple Linear Regression Equation¶

The equation of a straight line is:

$$ y = mx + b $$

Where:

  • m → slope (rate of change)
  • b → intercept (value when x = 0)
  • x → independent variable
  • y → dependent (predicted) variable

How Simple Linear Regression Works¶

The algorithm finds the best-fit line through data points by adjusting m (slope) and b (intercept)
to minimize the total squared distance between the points and the line — this is called
Ordinary Least Squares (OLS).

Function: linregress from scipy.stats¶

The linregress() function automatically computes:

  • slope (m)
  • intercept (b)
  • r-value (correlation coefficient)
  • p-value (significance test)
  • std_err (standard error of the estimate)

We’ll use these results to plot and interpret our regression line.
